## Setup Instructions

Follow these steps to set up and run the project locally:

1. **Clone the Repository**:
git clone

2. **Install Dependencies**:
npm install

3. **Run the Application**:
npm start

Reflection
This implementation of the Rock, Paper, Scissors game was an opportunity to practice React concepts such as states, components, and event handling. The game includes features like:
Dynamic updates based on user input.
A computer opponent with randomized choices.
A reset functionality to restart the game without refreshing the page.
